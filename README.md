# CodeCraft

CodeCraft is the name of my personal blog. This is created with the purpose of storing information and explanations of some of my projects in it, in order to have more accessibility and understanding of them over time.

## Dependencies

- "js-beautify": "^1.14.7"
- "react": "^18.2.0"
- "react-dom": "^18.2.0"
- "react-icons": "^4.7.1"
- "react-router-dom": "^6.6.2"
- "react-syntax-highlighter": "^15.5.0"

## How to run

To run the app, use the following command: "npm start"

To build the app, use the following command: "npm run build"
